# Role-Based Access Control (RBAC) Lab

## What I’m learning
How organizations control access to sensitive systems and data.

---

## Scenario
A retail company needs to restrict access to customer data.

---

## Roles

| Role | Access Level |
|------|-------------|
| Admin | Full access |
| HR | Employee data only |
| Support | Limited customer info |
| Guest | No sensitive access |

---

## Rules

- Employees only access data needed for their job
- Admins manage permissions
- Sensitive data is restricted

---

## Why this matters
RBAC reduces risk of insider threats and data leaks.

---

## Skills learned
- Access control
- Least privilege principle
- PII protection
